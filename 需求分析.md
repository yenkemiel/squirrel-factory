# 1. 主題概述
## 作品名稱
### **squirrel-factory** 是提供家用食物管理的系統
## 設計理念
> 提供一般家庭使用的簡易食物庫存表。家庭成員在市場採買前，可快速瀏覽家裡現有食物庫存以及快要到期的食物，加上快速檢視食物的營養資訊功能，為家裡的冰箱採購計畫以及當日餐桌提供更多的想法。

# 2. 功能
## 會員管理
1. 登錄 / 註冊
2. 對用戶的CRUD
## 食物管理
1. 對食物的CRUD
2. 資料匯出(CSV)
3. 過期通知(開啟頁面視窗)
4. 食評回饋
5. 常買食物功能
